# Changelog

## [0.1.0]

First release of the Enthropic specification.

- Two primitives: CONTEXT and CONTRACTS.
- Four first-class derived constructs: PROJECT, VOCABULARY, LAYERS, FLOWS.
- Formal EBNF grammar.
- Project file system: `enthropic.enth`, `state_[name].enth`, `vault_[name].enth`.
- Validation rules.
- Examples: e-commerce shop, CNC controller, offline-first notes app.
